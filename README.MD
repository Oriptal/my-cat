# my-cat

Небольшая учебная реализация `cat` на C для Linux.

Проект написан с использованием низкоуровневых POSIX-вызовов:

- `open`
- `read`
- `write`
- `close`

## Сборка

```bash
cc -Wall -Wextra -Wpedantic -std=c23 -o my_cat main.c
```

## Запуск

Без аргументов программа читает из стандартного ввода:

```bash
./my_cat
```

Чтение из файла:

```bash
./my_cat file.txt
```

Чтение из нескольких файлов:

```bash
./my_cat file1.txt file2.txt
```

Специальный аргумент `-` означает `stdin`:

```bash
printf 'hello\n' | ./my_cat -
```

## Поддержанные флаги

- `-n` — нумеровать все строки
- `-b` — нумеровать только непустые строки
- `-s` — сжимать подряд идущие пустые строки
- `-E` — показывать конец строки как `$`

Поддерживается группировка коротких флагов:

```bash
./my_cat -nE file.txt
```

После `--` все последующие аргументы считаются именами файлов:

```bash
./my_cat -- -n
```

## Примеры

```bash
printf 'a\n\nb\n' | ./my_cat -n
printf 'a\n\n\nb\n' | ./my_cat -s
printf 'a\nb\n' | ./my_cat -E
```

## Замечания

Это учебный проект, а не полный клон GNU `cat`.
Реализована только часть флагов и базовая обработка ошибок.
